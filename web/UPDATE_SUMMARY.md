### 1. 추천 페이지에서 장소 추가 시 예산/소개 표시 문제 수정 ✓

**문제**: 추천 페이지에서 장소를 선택하여 일정에 추가할 때 예산과 한줄 소개가 표시되지 않음

**원인**: `makePlaceCard` 함수에서 `isSelected` 체크 시 `state.selectedItems`가 객체 배열인데 `includes()`로 ID만 비교

**해결**:
```javascript
// 수정 전
const isSelected = state.selectedItems.includes(place.id);

// 수정 후
const isSelected = state.selectedItems.some(item => item.id === place.id);
```

**파일**: `app-simple.js` (약 520번째 줄)

### 3. AI 챗봇 일정표 기반 위치 중심 추천 기능 개선 ✓

**기능**: AI 챗봇이 현재 일정표의 방문 지역을 분석하여 해당 지역 중심으로 장소 추천

#### 개선된 로직

##### 1. 일정표 데이터 추출
```javascript
// 방문 지역 추출
const visitedCities = new Set();  // ["제주시", "서귀포시"]

// 방문 장소 추출
const visitedPlaces = [
  { name: "협재해수욕장", city: "제주시", category: "attraction", day: 1 },
  { name: "오션뷰 카페", city: "제주시", category: "cafe", day: 1 }
];
```

##### 2. 지역 기반 검색
- **특정 지역 언급**: "제주 카페" → 제주 지역만 검색
- **지역 미언급**: "맛집 추천" → 일정의 모든 지역에서 검색
- **중복 제외**: 이미 일정에 있는 장소는 제외

##### 3. 컨텍스트 기반 응답
```
사용자: "맛집 추천해줘"
AI: "제주시 지역의 5개 장소를 찾았어요! 일정 지역 내에서 추천해드려요."
```
