# 일정 페이지 업데이트 로그

## 📅 2026-06-01

### ✨ 새로운 기능

#### 1. 하루 3개 이상 장소 추천
- AI 일정 생성 시 하루에 최소 3개, 최대 4개의 장소 추천
- 첫날/마지막날: 3개
- 중간 날짜: 4개
- 카테고리별 균형있게 배치 (관광지, 카페, 맛집, 액티비티)

#### 2. 장소별 예산 및 한줄 소개 추가
- **예산 표시**:
  - 입장료가 있는 카테고리만 표시 (관광지, 액티비티, 숙소)
  - 음식점/카페는 예산 표시 안 함
  - 인원 수에 따라 총 예산 자동 계산
  - 예시: "💰 30,000원 (1인 15,000원)"

- **한줄 소개**:
  - 각 장소마다 간단한 설명 표시
  - AI 추천: 장소 설명 또는 태그 기반
  - 직접 입력: 사용자가 입력한 소개글

#### 3. 플러스/마이너스 버튼으로 장소 관리
- **➕ 버튼**: 해당 위치에 새 장소 추가
  - Day 헤더의 ➕: 해당 날짜 끝에 추가
  - 각 장소의 ➕: 해당 장소 위에 추가
- **➖ 버튼**: 장소 삭제
- 추천 페이지에서 선택 또는 직접 입력 가능

#### 4. AI 챗봇 추천 기능
- **위치**: 일정 페이지 > "🤖 AI 추천 받기" 버튼
- **기능**:
  - 자연어로 장소 검색 (예: "제주도 힐링 카페 추천해줘")
  - 실시간 채팅 인터페이스
  - 추천 장소 다중 선택
  - 선택한 장소를 원하는 날짜에 추가
- **현재 상태**: UI 완성, API 연동 대기 중

#### 직접 입력 시 동작
1. ➕ 버튼 클릭
2. "취소" 선택 (직접 입력)
3. 장소명 입력
4. 한줄 소개 입력 (선택)
5. 카테고리 선택 (1-5)
6. 예산 입력 (입장료가 있는 경우만)

2. "확인" 선택 (추천 페이지)
3. 추천 페이지로 이동
4. 장소 다중 선택
5. "일정에 추가" 클릭
6. 자동으로 예산, 설명 포함
